# 02 - 综合评级与三重共振

## 综合评级

### 评级流程

```
stock_signals（每只股票最多 7 条信号）
    ↓
加权混合评分
    ↓
评级映射（A+ / A / B / C / D）
    ↓
三重共振检测
    ↓
仓位计算
    ↓
integrated_ratings 表
```

### 评分公式

每个信号源的混合分：

```
blended = signal_direction × 60% + score × 40%

其中 signal_direction:
  bullish  = 100
  neutral  = 50
  bearish  = 0
```

综合加权分：

```
weighted_score = Σ(weight[source] × blended[source]) / Σ(weight[source])
```

**权重分配**：

| 信号源 | 权重 | 定位 |
|--------|------|------|
| valuation | 30% | 估值是否合理 |
| screener | 25% | 财务质量是否过硬 |
| buffett | 20% | 是否符合价值投资标准 |
| munger | 15% | 是否具备长期竞争力 |
| chan | 10% | 技术面是否支持 |

### 评级阈值

| 评级 | 分数 | 含义 |
|------|------|------|
| A+ | ≥ 85 | 极度看多，多维度强烈一致 |
| A | ≥ 70 | 看多，多数模型认可 |
| B | ≥ 55 | 中性偏多，信号不一致 |
| C | ≥ 40 | 中性偏空 |
| D | < 40 | 看空，多数模型不认可 |

### 评级分布示例（2026-03-30）

```
A  级:    42 只 ( 0.8%)  ← 候选买入池
B  级:   188 只 ( 3.8%)
C  级:   695 只 (13.9%)
D  级: 4,087 只 (81.5%)  ← 绝大多数不达标
```

## 三重共振

三重共振是系统的核心信号验证机制，要求**价值面、技术面、情绪面**三个维度同时看多（或看空），过滤掉单一维度的噪音。

### 买入共振（同时满足）

| 维度 | 条件 |
|------|------|
| 价值面 | valuation = bullish **或** buffett = bullish **或** munger = bullish |
| 技术面 | chan = bullish **且** score ≥ 55 |
| 情绪面 | trendradar = bullish **且** confidence ≥ 0.6 |

### 卖出共振（同时满足）

| 维度 | 条件 |
|------|------|
| 价值面 | valuation = bearish |
| 技术面 | chan = bearish |
| 情绪面 | trendradar = bearish |

### 共振效果

三重共振的意义在于**降低误判率**：
- 单一信号看多 → 可能是噪音
- 两个维度看多 → 有参考价值
- 三个维度同时看多 → 高置信度买入信号

## 仓位计算

```
position_pct = risk_manager_limit × rating_coefficient

rating_coefficient:
  A+ = 1.0（满仓位）
  A  = 0.8
  B  = 0.5
  C  = 0.3
  D  = 0.0（不持仓）
```

`risk_manager_limit` 由 Risk Manager 适配器基于波动率计算，高波动股票自动降低仓位上限。

---

## 数据存储

评级结果写入 `integrated_ratings` 表：

| 字段 | 类型 | 说明 |
|------|------|------|
| code | VARCHAR(10) | 股票代码 |
| date | DATE | 评级日期 |
| rating | VARCHAR(5) | A+ / A / B / C / D |
| weighted_score | FLOAT | 0-100 综合分 |
| resonance_buy | BOOLEAN | 三重共振买入标记 |
| resonance_sell | BOOLEAN | 三重共振卖出标记 |
| position_pct | FLOAT | 建议仓位百分比 |
| detail_json | JSONB | 各信号源明细 + 共振原因 |
