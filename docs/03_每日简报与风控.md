# 03 - 每日简报与持仓风控

## 每日简报

简报是系统的核心输出，汇总当日所有分析结果，保存为 `reports/briefing_YYYY-MM-DD.md`。

### 简报结构

```
# 每日投资决策简报 2026-03-30

## 三重共振买入预警          ← 最高优先级信号
## 三重共振卖出预警
## 综合评级变动              ← 升级/降级跟踪
## 候选池变动 (A+/A级)       ← 新入池/出池
## 持仓风控提醒              ← 对当前持仓的风险预警
## 持仓深度分析              ← 盈亏、信号明细、操作建议
## 综合评级概览              ← A+/A/B/C/D 分布
## Top 20 优选股票           ← A 级股票排行
```

### 各板块详情

#### 三重共振预警

列出所有触发三重共振的股票，包含：
- 股票代码和名称
- 综合评分
- 建议仓位
- 触发原因（价值面看多 + 缠论买入 + 板块利多）

#### 评级变动

对比今日与前一交易日的评级，列出：
- 升级股票：如 B → A、C → B
- 降级股票：如 A → B、B → C

#### 候选池变动

追踪 A+/A 级股票池的变化：
- 新入池：首次进入 A 级的股票
- 出池：从 A 级降级的股票
- 池内总数

## 持仓风控

### 数据来源

持仓数据从 `portfolio.csv` 读取：

```csv
code,name,shares,cost_price,buy_date,notes
002007,华兰生物,900,15.793,2025/10/10,
600600,青岛啤酒,200,68.665,2025/7/31,
002179,中航光电,300,34.757,2026/2/6,
```

### 风控提醒

对每只持仓股票检查：
- 是否有 bearish 信号（buffett/munger/valuation/chan 看空）
- 缠论是否出现卖出信号
- 所属板块舆情是否利空

### 持仓深度分析

| 分析项 | 内容 |
|--------|------|
| 组合概览 | 持仓数、总成本、总市值、浮动盈亏 |
| 持仓明细 | 每只股票的成本价、现价、盈亏%、持有天数、评级 |
| 信号明细 | 每只股票的看多/看空信号列表 |
| 风险提示 | 看空信号 ≥ 3 个的持仓 |
| 操作建议 | 基于评级和信号给出持有/减仓/观察建议 |
