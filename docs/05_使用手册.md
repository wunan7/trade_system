# 05 - 使用手册

## 环境要求

- Python 3.11+
- PostgreSQL（finance 库 + finance_public_opinion 库）
- 依赖子项目：a-stock-screener / a-stock-valuation / a-stock-analysis / a-stock-chan / TrendRadar / finance_data

## 快速开始

### 首次安装

```bash
cd C:\Users\wunan\projects\legacy_solution_integration

# 安装依赖
pip install -r requirements.txt

# 创建数据库表
python -m src migrate
```

### 日常分析（每日运行）

```bash
# 方式一：全流程一键运行
python -m src run

# 方式二：分步运行
python -m src collect                    # 采集全部信号
python -m src rate                       # 综合评级
python -m src briefing --dry-run         # 生成简报（不推送）
```

简报自动保存到 `reports/briefing_YYYY-MM-DD.md`。

## 全部命令

### 数据采集

```bash
# 采集所有信号源
python -m src collect

# 采集指定信号源
python -m src collect --source screener
python -m src collect --source chan

# 采集指定日期
python -m src collect --date 2026-03-30
```

### 综合评级

```bash
python -m src rate                       # 今日评级
python -m src rate --date 2026-03-30     # 指定日期
```

### 每日简报

```bash
python -m src briefing --dry-run         # 生成但不推送
python -m src briefing                   # 生成并推送
python -m src briefing --date 2026-03-30 # 指定日期
```

### 策略回测

```bash
# 列出所有策略
python -m src list-strategies

# 回测（默认 3 年，default 策略）
python -m src advanced-backtest

# 指定策略和日期范围
python -m src advanced-backtest --strategy aggressive --start 2023-03-27 --end 2026-03-27
```

### 历史信号预计算

```bash
# 完整预计算（含缠论）
python -m src precompute --start 2023-03-27 --end 2026-03-27

# 快速预计算（跳过缠论）
python -m src precompute --start 2023-03-27 --end 2026-03-27 --skip-chan

# 只计算某个信号源
python -m src precompute --source valuation --start 2023-03-27 --end 2026-03-27
```

### 信号级回测

```bash
python -m src backtest                                    # 默认
python -m src backtest --start 2026-03-01 --end 2026-03-29 --holding-days 5,10,20
```

## 创建自定义策略

1. 复制一个现有策略文件：
```bash
cp strategies/default.json strategies/my_strategy.json
```

2. 编辑参数（如调整权重、止损线等）

3. 回测验证：
```bash
python -m src advanced-backtest --strategy my_strategy --start 2023-03-27 --end 2026-03-27
```

4. 报告自动保存到 `reports/backtest_my_strategy_{时间}.md`

## 持仓管理

编辑 `portfolio.csv`，格式如下：

```csv
code,name,shares,cost_price,buy_date,notes
002007,华兰生物,900,15.793,2025/10/10,
600600,青岛啤酒,200,68.665,2025/7/31,
```

更新后重新运行 `python -m src briefing --dry-run` 即可看到最新的持仓风控分析。
