## 三、核心发现

### 3.1 有效的优化方向

| 优化方向 | 单独收益提升 | 叠加后收益 | 关键机制 |
|---------|------------|-----------|---------|
| 动态止损止盈 | +3.66% | +25.24% | ATR 自适应止损 + 移动止盈 + 时间止损 |
| 宏观择时 | +2.80% | +28.04% | 沪深300趋势 + 市场宽度 + 波动率 |
| 信号一致性（温和） | +0.95% | +28.99% | 5个信号源一致看多时加仓20% |

### 3.2 无效或负面的优化方向

| 优化方向 | 收益影响 | 失败原因 |
|---------|---------|---------|
| 相关性约束 | -5.68% | A股同质性高，严格的相关性阈值导致持仓断崖式下降，错失收益 |
| 行业集中度上限 | -3.63% | 回撤仅改善 0.3%，但损失大量收益，限制了强势行业配置 |
| 黄金坑试探 | -2.01% | 技术面看空的低估股票短期确实还在跌，额外买入拖累了整体表现 |
| 多层级共振 | -1.97% | 强共振难触发，总体上拉低了原先激进的 1.5x 加仓乘数 |
| 自适应权重 | -19.08% | 历史准确率的滞后性，市场风格轮动时失效 |
| 仓位集中度（激进） | -17.65% | 乘数叠加过高（1.3×1.2=1.56x），过度集中 |
| 分批建仓 | -20.97% | 月度检查点下永远半仓运行，无法满仓 |
| 周频风控 | -31.44% | ATR/trailing stop 参数针对月频校准，周频触发过多误杀 |
| 多功能全开 | -6.90% | 自适应权重 + 宏观择时双重减仓，过度保守 |

### 3.3 关键教训

1. **简单有效 > 复杂精巧**：concentrated_v2（3 层优化）优于 ultimate（全开）
2. **滞后指标有害**：历史准确率、自适应权重在风格轮动时会失效
3. **参数要匹配频率**：止损参数是针对月频校准的，直接切周频会过度止损
4. **乘数叠加要克制**：多个 1.2-1.3x 乘数叠加后的极端值会导致过度集中
5. **当前状态 > 过去表现**：信号一致性（当前 5 个信号怎么看）比历史准确率（过去 6 个月怎么样）更有价值
6. **过度风控会扼杀收益**：相关性约束、行业集中度限制虽然听起来合理，但在实际多头组合中会导致资金闲置。

## 四、最终推荐

### 生产环境推荐 1：单策略王者 concentrated_v2

追求高额超额收益，愿意承受约 -23% 的中等回撤。

**配置要点**：
| 模块 | 状态 | 参数 |
|------|------|------|
| 动态止损止盈 | ✅ 开启 | ATR 2.0x, trailing 15%, 时间止损 6 月 |
| 宏观择时 | ✅ 开启 | risk_on 1.2x, risk_off 0.5x |
| 信号一致性 | ✅ 开启 | 5看多 1.2x, 4看多 1.1x, ≤2看多 0.9x |
| 流动性折扣 | ✅ 开启 | <5000万 0.9x, <1000万 0.8x |
| 历史准确率 | ❌ 禁用 | 设为 1.0（避免滞后性） |
| 自适应权重 | ❌ 禁用 | 过度保守 |
| 分批建仓 | ❌ 禁用 | 月频不适配 |
| 相关性约束 | ❌ 禁用 | 收益惩罚过高 |

### 生产环境推荐 2：机构级防守 tactical 多策略组合

追求极端的下行风险保护，最大回撤控制在 -15% 级别。

**核心机制**：
- 牛市重仓 concentrated_v2
- 当检测到宏观 `risk_off` 时，强制从主策略抽离 80% 资金
- 转移给 `defensive` 防御策略（专买高股息、极度低估、严格 -5% 止损的大蓝筹）
- 结果：总收益 +17.44%，最大回撤缩减至惊人的 **-15.83%**。
