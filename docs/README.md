# A 股智能交易决策系统

## 系统概述

本系统整合 7 个独立分析子系统（财务筛选、估值分析、巴菲特模型、芒格模型、缠论技术面、舆情分析、风控管理），通过多模型加权评级 + 三重共振验证，为 A 股投资提供系统化的选股、评级、风控和回测能力。

## 核心架构

```
┌─────────────────────────────────────────────────────────┐
│                     数据采集层                            │
│  screener │ valuation │ buffett │ munger │ chan │ 舆情 │ 风控  │
└─────────┬───────────┬─────────┬────────┬──────┬──────┬──┘
          └───────────┴─────────┴────────┴──────┴──────┘
                              ↓
                    ┌─────────────────┐
                    │  stock_signals  │  ← 统一信号表
                    └────────┬────────┘
                             ↓
                    ┌─────────────────┐
                    │   综合评级引擎    │  ← 加权评分 + 三重共振
                    └────────┬────────┘
                             ↓
                    ┌─────────────────┐
                    │integrated_ratings│ ← 评级结果表
                    └────────┬────────┘
                             ↓
              ┌──────────────┼──────────────┐
              ↓              ↓              ↓
        ┌──────────┐  ┌──────────┐  ┌──────────┐
        │ 每日简报  │  │ 持仓风控  │  │ 策略回测  │
        └──────────┘  └──────────┘  └──────────┘
```

## 技术栈

- **语言**: Python 3.11
- **数据库**: PostgreSQL（finance 库 + finance_public_opinion 库）
- **ORM**: SQLAlchemy
- **子系统**: a-stock-screener / a-stock-valuation / a-stock-analysis / a-stock-chan / TrendRadar / finance_data

## 数据库

| 数据库 | 用途 | 核心表 |
|--------|------|--------|
| finance | 行情 + 财务 + 信号 + 评级 | stock_daily, financial_summary, stock_signals, integrated_ratings |
| finance_public_opinion | TrendRadar 舆情 | ai_analysis_results |
